Mastermind

Single player VS CPU

Object

Duplicate the exact color and position of your opponent's hidden pegs in the least number of moves

How to Play

A secret code is selected by your opponent, the codemaker, using any combination of 6 colored code pegs:

You have 10 chances to guess the secret code. After each guess, you are given feedback in the form of key pegs to the right.

Black key peg: A code peg is the correct color and in the correct spot

White key peg: A code peg is the correct color, but in the incorrect spot
